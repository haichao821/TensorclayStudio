# TensorClay 专注土的本构理论研究的单元试验计算软件

Tensorclay是一款简单的用于研究土的本构理论的单元试验计算软件，可以模拟排水和不排水三轴试验。主要特点为可以同步对比土的三轴压缩试验结果和模型计算结果。

目前，我们开放共享Tensorclay的执行文件，后期将逐步补充经典的土的本构模型，实现更多的单元试验类型和不同加载条件，如温度和基质吸力控制等。

Tensorclay仍处于快速迭代开发过程，程序界面和功能可能在不同版本之间会存在差异，欢迎提出宝贵建议。

## Tensorclay-R2026M07_beta版本更新 - 2026年07月14日
本次更新对底层架构进行了深度调整，重构了界面布局与交互流程，旨在降低长时间使用的信息密度与视觉疲劳，同时大幅提升了三轴试验数据处理与单元计算的效率。
### 📑 功能精简与聚焦
- 【移除】正式删除 EIDIPAD（文本编辑）与 PAPERS（文献管理）功能页，不再维护非核心的文本与文献管理功能。
- 【聚焦】核心精力将全面集中于 KANBAN（看板）、TRIAX（三轴）和 MANUAL（手册）三大功能页。

### 🎨 界面布局与视觉重构
【新增】界面最上方增加独立的一行，用于全局展示当前的项目名称。

【优化】菜单栏拆分布置：
- 将“打开项目”、“保存项目”、“计算”、“中止”等核心控制按键迁移至左上角独立区域。
- 将“Add Case”、“Add Plot”等操作按钮移至左侧项目树顶部，并简化为纯图标显示。
- 【优化】降低信息密度：大幅精简右侧面板的文字内容，增加区域留白，有效缓解长时间科研工作带来的视觉疲劳。
- 【优化】右侧面板拆分为两部分：
	- 上半部分：作为中间面板的随动区间，动态补充展示所选本构模型、初始条件和实验类型的相关信息。
	- 下半部分：专用于输出内置求解器的实时计算运行信息。
### 🔄 Case 节点交互逻辑优化
- 【重构】取消早期版本点击项目树 Case 节点时弹出的操作窗。
- 【优化】新交互改为：点击左侧项目树的 Case 节点时，中间面板与右侧面板同步自动刷新。
	- 中间面板直接展示对应的 .inp 文件内容。
	- 原本的弹窗操作选项被转化为局部菜单栏，固定集成在中间面板上方（支持一键导入内置求解器计算、重置参数等）。
### 📊 DATA 节点与数据导入重做（TRIAX 模式）
- 【新增】扩展三轴试验数据兼容性，DATA 节点支持直接读取 GDS 等主流三轴设备采集的原始试验结果文件。
- 【新增】增加“Open”功能按钮用于导入特定格式的数据文件，导入后可自动将其转换为 Tensorclay 内置的 .sdb 数据格式。
- 【新增】中间面板下部区域增加表格（Table）预览功能，支持对导入的数据进行直观的预览。

- 对程序界面进行改进，精简了最下方的状态栏，同时对左侧的模式切换按钮进行微调，如下图所示。

- 优化了计算模式的代码，删减了不必要的操作节点。在当前版本中，Source操作节点用于提取模型预测结果和三轴试验结果。Layout和Figure操作节点主要终于设置Gnuplot的绘图参数，ARTIST操作节点用于数据采样和单位变换。借助该绘图模式可以一键生成高质量的矢量图。
